Theme devs
==========

Want to hack on the css/display of the site? Do this stuff.

[install poetry](https://poetry.eustace.io/docs/)

`poetry install` in checked out source.

make your theme hacks

`poetry run make devserver` (or similar)

navigate to local compiled site

commit and push work to github branch and/or open a PR


Article/Content Editing
=======================

Edit the markdown files, send to github. Optionally you can render it locally.

You can edit the markdown from the web.

Pelican's docs on editing https://docs.getpelican.com/en/stable/content.html
